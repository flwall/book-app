A Book Management App with a Quarkus REST-API (J2EE) (and a Apache Derby Embedded DB in the Background)

And a React and Android Frontend

  

# Installation

## start Server

 ``cd book_api``
``mvnw quarkus:dev``

## start web-frontend
 ``cd bookmgmnt-react-front`` 
``npm i`` and ``npm start``
